JavaScript Operators Chart
No. Operator Type Use / Example
1 Grouping ( ) — Groups expressions, To control the order of operations (what runs first)., e.g., (2 + 3) * 4 = 20
2 Exponentiation ** —To raise a number to a power. e.g., 2 ** 3 = 8
3 Multiplication * To multiply two numbers.— e.g., 4 * 5 = 20
4 Addition + To add numbers or combine text (strings) — e.g., 10 + 5 = 15 , ➤ Example (numbers): 10 + 5 = 15 ➤ Example (strings): 'Hi' + ' Nazia' = 'Hi Nazia'
5 Relational <, > ➤ Use: To compare values (greater than or less than).— e.g., 5 < 10 → true , ➤ Example: 5 < 10 → true, 7 > 3 → true
6 Equality ==, === == checks values only (ignores type) === checks value and type both— e.g., 5 == '5' → true, 5 === '5' → false , 5 == '5' → true (value same) 5 === '5' → false (type different)
7 Logical AND && Returns true only if both sides are true— e.g., true && false → false , true && true → true, true && false → false
8 Logical OR || Returns true if any one side is true.— e.g.,true || false → true, false || false → false false || true → true
9 Conditional (Ternary) condition ? expr1 : expr2 Short way to write if...else statements — e.g., age > 18 ? 'Adult' : 'Minor' , age > 18 ? 'Adult' : 'Minor' If age is more than 18 → shows 'Adult', otherwise → 'Minor'
10 Assignment =, +=, *= To give a value to a variable or update it— e.g., x = 10, x += 5 , x = 10 (assigns 10) x += 5 (means x = x + 5) x *= 2 (means x = x * 2)
